# Design Tokens

Design tokens are the visual design atoms of our design system. They store visual design attributes like colors, typography, spacing, and more.

## Color Palette

Our color system includes semantic colors for different states and contexts, as well as neutral grays for light and dark modes.

## Usage in SwiftUI

```swift
import SwiftUI
import GlideComponents

struct ExampleView: View {
    var body: some View {
        VStack(spacing: 16) {
            Text("Welcome")
                .font(GlideTypography.titlePage)
                .foregroundColor(GlideColors.textPrimary)
            
            GlideCard {
                VStack(alignment: .leading, spacing: 12) {
                    Text("Card Content")
                        .font(GlideTypography.heading)
                        .foregroundColor(GlideColors.textPrimary)
                    
                    Text("This card uses our design tokens for consistent styling.")
                        .font(GlideTypography.body)
                        .foregroundColor(GlideColors.textSecondary)
                }
            }
        }
    }
}
```

## Usage in CSS

All design tokens are available as CSS custom properties:

```css
.title {
  font-size: var(--typography-title-page-font-size);
  font-family: var(--typography-title-page-font-family);
  color: var(--color-gray-light-900);
}

.card {
  background-color: var(--color-gray-light-25);
  border: 1px solid var(--color-gray-light-200);
  border-radius: 12px;
}
```
